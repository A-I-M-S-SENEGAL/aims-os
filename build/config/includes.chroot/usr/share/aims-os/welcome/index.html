AIMS

Bienvenue dans AIMS OS

Système éducatif basé sur Debian 13 — African Institute for Mathematical Sciences, Sénégal

Ce qui est pré-installé

AIMS OS v2.0 a été calibré sur les 45 cours du curriculum 2025-2026. La liste ci-dessous résume les piliers ; pour le détail, voir le projet upstream sur GitHub.

Mathématiques & science

NumPy, SciPy, SymPy, pandas, matplotlib, scikit-learn, R + tidyverse, Octave, Maxima, GeoGebra, TeX Live, Jupyter.

Big Data & IA

Dask + distributed, NLTK, OpenCV, xarray, netCDF, scikit-image, mpi4py, biopython. PyTorch / JAX / Polars / DuckDB via pipx ou mamba.

Dev Web & Android

Node 22 LTS, npm, Bun, Deno, OpenJDK 21, Gradle, Kotlin, adb, fastboot, PHP, Composer.

Conteneurs & cloud

Docker Engine + Compose v2 + Buildx, Podman, kubectl, Ansible.

IDEs & éditeurs

VSCodium (libre), Cursor (AI), RStudio, neovim, GNOME Text Editor.

Bases de données

DBeaver CE + clients PostgreSQL, MariaDB, Redis (serveur local), MongoDB, SQLite.

Sécurité & forensics

nmap, Wireshark, tcpdump, John the Ripper, hashcat, aircrack-ng, Sleuth Kit, foremost, binwalk, scapy.

Bureautique

LibreOffice complet (FR + EN), Firefox ESR, Chromium, Evolution, GIMP, Inkscape, VLC.

Pour aller plus loin

Pour installer ce qui n'est pas inclus :

# Paquet Debian
sudo apt install <paquet>

# Outil Python isolé (ruff, jupyterlab, sagemath, …)
pipx install <outil>

# Environnement scientifique reproductible
mamba create -n monenv python=3.13 pytorch jupyterlab

# App graphique sandboxée
flatpak install flathub <app-id>

Welcome to AIMS OS.

  Built at the African Institute for Mathematical Sciences — Senegal.
  Debian 13 (Trixie) base — GNOME desktop — scientific stack pre-loaded.

What's here
-----------
  Python   numpy, scipy, pandas, matplotlib, sympy, scikit-learn, JupyterLab
  Math     Maxima, Octave, GeoGebra, gnuplot (SageMath via `mamba install`)
  R        r-base + tidyverse + rmarkdown
  LaTeX    TeX Live full + texstudio + biber + French language packs
  Conda    `mamba` and `conda` (Miniforge3 installed under /opt/miniforge3)

How to install more
-------------------
  Debian:    sudo apt install <package>
  Flathub:   flatpak install flathub <app-id>
  Python:    pipx install <tool>         (or use `mamba create -n …`)
  R:         install.packages("<pkg>")   (inside R)

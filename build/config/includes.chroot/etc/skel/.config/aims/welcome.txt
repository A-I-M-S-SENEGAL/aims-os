Welcome to AIMS OS.

  African Institute for Mathematical Sciences — Sénégal.
  Debian 13 (Trixie) base · GNOME 48 · curriculum 2025-2026 calibrated.

What's pre-installed
--------------------
  Math       NumPy, SciPy, SymPy, pandas, matplotlib, scikit-learn,
             R + tidyverse, Octave, Maxima, GeoGebra, TeX Live, Jupyter
  Big Data   Dask + distributed, NLTK, OpenCV, xarray, netCDF, scikit-image,
             mpi4py, biopython; DBeaver, Postgres/MariaDB/Redis/MongoDB
  Security   nmap, Wireshark, tcpdump, John, hashcat, aircrack-ng,
             Sleuth Kit, foremost, binwalk, scapy, cryptography
  Dev Web    Node 22 LTS, npm, Bun, Deno, OpenJDK 21, Gradle, Kotlin,
             PHP, Composer, adb, fastboot
  Containers Docker + Compose v2 + Buildx, Podman, kubectl, Ansible
  IDEs       VSCodium, Cursor, RStudio, neovim
  Office     LibreOffice (FR + EN), Firefox ESR, Chromium, GIMP, Inkscape

How to install more
-------------------
  Debian    sudo apt install <package>
  Python    pipx install <tool>          (ruff, jupyterlab, sagemath, …)
  Conda     mamba install -c conda-forge <pkg>
  Flathub   flatpak install flathub <app-id>
  R         install.packages("<pkg>")    (from inside R)

On an existing Debian, add the AIMS OS apt repo:
  https://a-i-m-s-senegal.github.io/aims-os/

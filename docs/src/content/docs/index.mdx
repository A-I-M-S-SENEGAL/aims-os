## Ce que vous obtenez

AIMS OS regroupe sous une seule installation la stack scientifique
nécessaire au curriculum (NumPy, SciPy, R, Octave, Maxima, TeX Live,
Jupyter, ...), la couche développeur moderne (Node 22 LTS, Cursor,
VSCodium, RStudio, Docker, Bun, Deno, uv, ...) et un bureau GNOME 48
en français, le tout vérifié sur arm64 (Mac M-series via UTM) et amd64
(PC portables des labs).

L'installation prend une demi-heure en partant d'une clé USB ou d'une
VM. La couche dev tient déjà debout après le premier reboot, sans
configuration supplémentaire.

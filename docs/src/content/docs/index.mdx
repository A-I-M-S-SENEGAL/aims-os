AIMS OS est un système d'exploitation conçu pour les étudiants de
l'**African Institute for Mathematical Sciences Sénégal**. Base
**Debian 13 (Trixie)**, GNOME 48 en français, et la stack
scientifique et développeur préparée pour les 45 cours du
curriculum 2025-2026.

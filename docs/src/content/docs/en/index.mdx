## What you get

AIMS OS bundles, under a single install, the scientific stack the
curriculum needs (NumPy, SciPy, R, Octave, Maxima, TeX Live, Jupyter,
...), the modern developer layer (Node 22 LTS, Cursor, VSCodium,
RStudio, Docker, Bun, Deno, uv, ...) and a French-localised GNOME 48
desktop, all verified on arm64 (Mac M-series via UTM) and amd64
(lab laptops).

The install takes about half an hour from a USB stick or a VM. The
dev layer is already up after the first reboot, no extra config.

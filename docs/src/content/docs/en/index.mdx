AIMS OS is an operating system built for students at the
**African Institute for Mathematical Sciences Senegal**.
Debian 13 (Trixie) base, French-localised GNOME 48, plus the
scientific and developer stack tuned to the 45 courses of the
2025-2026 curriculum.
